Letter graveyard

Page overview

Brief

  • Make a game of Hangman
  • The guessable word should be randomly chosen from a list of pre-determined words (of different lengths)
  • Make a display areas for both the correct and incorrect guesses
  • Make a countdown of how many guesses the player has before they lose the game.
  • Make a win/lose state for the end of the game

Concepts used

  • Data handling; Using data from an HTML form, and using objects and arrays to hold/change the game's score/status
  • Practice using forEach and indexOf methods

Problems encountered

  • At the time I made it, this was the most complex project I'd ever attempted. It had multiple "phases", dynamic/random input and multiple possible end states. I had leared a lot of systems at but hadn't really needed to combine them in such a complex way before then.
  • I actually had to throw away my first attempt and start again but I feel like I learned a lot from it.
  • The problem that probably frustrated me most was dealing with duplicated letters in the answer word. Luckily I was eventually able to solve it with some clever loops.
