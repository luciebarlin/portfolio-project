Page overview

  • Brief
    • Make a small, scalable deck of cards that move in sequence using CSS and JS.
    • Bonus: make the cards move back to start when they have all moved left
    • Extra bonus: keep the final z-index of the cards
  • Concepts used
    • SCSS array, for loop is SCSS, nth-child, practice defining and using variables
    • Practice setTimeout function
  • Problems encountered
    • It was difficult at first to understand and apply the SCSS loop
    • I had to include some JS to slide away the page overview on load, otherwise the user would miss the first cards sliding
    • There were many calculations involved to achieve the correct x-offsets of the cards. This was difficult to visualise and a lot of trial and error occurred.
